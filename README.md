# Levels of Processing — Memory Experiment

This is a cognitive psychology experiment investigating how the way we process information affects how well we remember it. It is based on the **Levels of Processing** framework (Craik & Lockhart, 1972), which proposes that deeper, more meaningful processing of information leads to stronger memory traces.

## About the Experiment

This experiment investigates how different types of questions affect memory for images. It consists of two short phases and takes approximately 10 minutes.

## How to Participate

1. Open the experiment link in a desktop or laptop browser:
   https://jenzhee.github.io/levels-of-processing/index.html
3. Read the instructions carefully
4. For each image, answer the question shown by pressing:
   - **F = Yes**
   - **J = No**
5. Try to respond as quickly and accurately as possible
6. The experiment ends automatically and your results are recorded

## Notes

- Please complete the experiment in one sitting without interruptions
- Do not refresh or close the tab mid-experiment as your data will not be saved

## Details

Built with [jsPsych 7](https://www.jspsych.org/), an open-source JavaScript library for behavioral experiments. Data is collected anonymously and stored securely via [SheetDB](https://sheetdb.io/), which transmits responses to a private Google Sheet through JSON API. No personally identifiable information is recorded.

Stimuli are drawn from the [Snodgrass & Vanderwart (1980)](https://figshare.com/articles/dataset/Snodgrass_Vanderwart_Like_Objects/3102781?file=4822669) standardized set of line drawings, widely used in cognitive psychology research for their consistent naming, familiarity, and visual complexity across participants.
